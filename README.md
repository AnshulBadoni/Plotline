# 🎭 Plotline — Create Interactive Stories, No Code Required

> A modern, open-source visual novel and branching narrative builder — built for writers, educators, and indie game makers.

[![Next.js](https://img.shields.io/badge/Built%20with-Next.js-000000?style=for-the-badge&logo=nextdotjs)](https://nextjs.org)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg?style=for-the-badge)](LICENSE)
[![Local First](https://img.shields.io/badge/Storage-Local_First-purple?style=for-the-badge)](https://localfirstweb.com)

Plotline is a powerful, no-code platform for creating interactive stories, visual novels, and narrative games — without writing a single line of code.

Built with Next.js, it gives you a sleek, drag-and-drop editor, live preview, and full local data control. Whether you're crafting a fantasy RPG, a classroom simulation, or a choose-your-own-adventure novel, Plotline makes it fast, intuitive, and fun.

Your data never leaves your device.  
Export to HTML, JSON, or embed anywhere.  
Extend it. Customize it. Make it yours.

---

## 🌟 Why Plotline?

Most tools force you to choose:  
👉 Easy but technical (like Twine)  
👉 Powerful but technical (like Unity)  

Plotline bridges the gap:
- No-code editor — Drag, connect, and edit story nodes visually
- Branching logic — Add conditions, variables, and dynamic choices
- Live preview — See your story play out in real time
- Character & stat management — Track HP, relationships, inventory
- Media support — Add images, videos, and sound (coming soon)
- Local-first — All data stored in localStorage — private, safe, offline
- Exportable — Generate standalone HTML or JSON for sharing or publishing

No cloud. No tracking. No compromises.

## 🚀 Getting Started

1. Clone the project  
   Run these commands in your terminal:

   git clone https://github.com/yourusername/plotline.git
   cd plotline
   npm install
   npm run dev

2. Open your browser at [http://localhost:3000](http://localhost:3000) and start building.

---

## 🛠 Tech Stack

- Framework: Next.js (App Router, React Server Components)
- State: Custom hooks + reducer pattern (pluggable for Zustand/Pinia)
- UI: Tailwind CSS + Lucide icons
- Storage: localStorage (future: IndexedDB, file export)
- Architecture: Component-driven, modular, ready for plugins

Perfect for creators who want simplicity — and developers who want to extend it.

## 📦 Export & Share Your Story

Once your narrative is ready:
- Run `npm run export` → generates a standalone HTML file
- Export as JSON for version control or reuse
- Embed in websites, blogs, or LMS platforms

Great for:
- Writers publishing interactive fiction
- Teachers creating decision-based lessons
- Game devs prototyping story arcs
